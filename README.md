
# WhatsApp FCM Lab

A professional **WhatsApp-style Flutter application** demonstrating **Firebase Cloud Messaging (FCM)** push notifications — built for Year 3 CSE Lab 6.

## Features

* WhatsApp-style UI with Chats, Status, and Alerts tabs
* Push notifications via Firebase Cloud Messaging (FCM)
* Foreground notifications — in-app banner + popup dialog
* Background & terminated state notification handling
* Device token display & one-tap copy
* Notification history screen
* Dark mode support

## Getting Started

### Prerequisites

* Flutter SDK `>=3.0.0`
* Android Studio or VS Code
* A real Android/iOS device (FCM requires a real device)
* A Firebase account

### 1. Clone the repository

```bash
git clone https://github.com/SilasHakuzwimana/whatsapp_fcm_lab.git
cd whatsapp_fcm_lab
```

### 2. Install dependencies

```bash
flutter pub get
```

### 3. Firebase Setup

1. Go to [Firebase Console](https://console.firebase.google.com/)
2. Create a new project
3. Add an Android app with package name `com.example.whatsapp_fcm`
4. Download `google-services.json` and place it in `android/app/`
5. Run FlutterFire CLI:

```bash
dart pub global activate flutterfire_cli
flutterfire configure
```

### 4. Run the app

```bash
flutter run
```

> Must run on a **real device** — FCM does not work on emulators.

## How to Send a Test Notification

1. Run the app and tap the **key icon** (top right)
2. **Copy** your device token
3. Go to [Firebase Console](https://console.firebase.google.com/) → **Engage** → **Messaging**
4. Click **New campaign** → **Firebase Notification messages**
5. Enter a **Title** and **Body**
6. Click **Send test message**
7. Paste your token → **Add** → **Test**

## Android Configuration

Key settings in `android/app/build.gradle.kts`:

```kotlin

    isCoreLibraryDesugaringEnabled = true
    sourceCompatibility = JavaVersion.VERSION_17
    targetCompatibility = JavaVersion.VERSION_17
}

dependencies {
    coreLibraryDesugaring("com.android.tools:desugar_jdk_libs:2.1.4")
}
```
